With an intuitive quiz design editor and platform, anyone can create their very own dynamic quiz with automatic result calculations and respondent categorization. Then, share your "vibe check" with friends, sit back, and watch as the results come rolling in!

## Features

- Register an account and log in (w/ email & password)
- Create a quiz
- View, edit, and delete a quiz with a visual builder
- Take a quiz and receive personalized and ranked results

## Development

```bash
pnpm dev
```

Now your very own vibecheck development instance is running on localhost!
